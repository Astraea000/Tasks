# 一、程序用途

数学四则运算与字符串拼接功能

# 二、程序功能

1.两个浮点数相加

2.两个浮点数相减

3.两个浮点数相乘

4.两个浮点数相除

5.两个字符串拼接

6.非法选项容错提示

# 三、使用步骤

先运行，再根据友好菜单提示，输入数字选择对应功能，按照提示完成操作

# 四、相关知识点

`package main`：声明当前为可执行程序包

`func main()`：程序唯一入口函数，程序从这里开始执行

`import "fmt"`：导入标准输入输出包，用于控制台打印、读取输入

使用 `var` 关键字定义变量，用到 `float64`（浮点型）、`int`（整型）、`string`（字符串类型）

`fmt.Println()`：普通换行打印

`fmt.Printf()`：格式化输出

`fmt.Scanln()` / `fmt.Scan()`：读取控制台输入，结合取地址符 `&` 接收变量值

`switch`语句多分支判断，选择不同路线进行运行
