# 对Markdown的进一步了解

以下内容来源于markdown的使用以及资料的查找。



首先，对我而言，这是第一次使用markdown，它和word的区别在于，它没有那么多程序员不需要的功能，它同时又具有程序员需要的功能，这是我觉得markdown最有意思的地方。敲代码并不需要字体，也不需要太多的大小区分，更不需要代码变个颜色，markdown让人能够更加专注地投入到自己的工作中，不会因为功能复杂而分心。那些导出为pdf，HTML等等的功能，让人不得不喜欢使用它。



其次，markdown比word更好上手。因为内容简单，所以学习起来十分方便，基本上使用几次就能记住相关语法，而不是像word那样，考虑缩进，间距，页眉页脚等等问题，让我们回归到了内容本身。



对于typora来说，我觉得它的好用体现在简洁，它的界面真的超级超级超级干净，干净到我觉得没有一句废话。
